Toy example: Stein Variational Gradient Descent: A General Purpose Bayesian Inference Algorithm

Intro

Stein Variational Gradient Descent (SVGD) is a variational inference algorithms that by solving a deterministic optimisation problem seek to create a set of samples from an untraceable posterior distribution. More specifically SVGD tries to transform a set of initial random particles to match the posterior distribution.

You can read more in the original paper: https://arxiv.org/abs/1608.04471

I coded a toy example to understand SVGD deeper. You can find the code here https://github.com/jkkronk/toy_svgd/blob/main/toy_svgd.ipynb or use the google colab to play around with hyper parameters here https://colab.research.google.com/drive/1FPAkreW7eGq_CUG3slAgNtUODD5rTozA#scrollTo=_Ua9iOMMVnY-

Enjoy!
